# 🛒 Amazon Egypt Price Tracker Bot 🚀

A Python-based Telegram bot that monitors Amazon Egypt product prices and sends real-time notifications when prices drop.

## ✨ Features
* **Live Tracking:** Monitors price changes for any Amazon.eg product.
* **Instant Alerts:** Sends a Telegram message as soon as a price drop is detected.
* **Database Support:** Efficiently manages tracked products using SQLite.
* **Easy Setup:** Configurable via environment variables.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Libraries:** Selenium, BeautifulSoup4, Requests.
* **Database:** SQLite.
* **Tools:** Git, Linux (Fedora).
